# Assignment 2 Portfolio Website

## Project Description
This project is an improved version of my Assignment 1 portfolio website. It keeps the original portfolio idea, then adds interactivity and better user experience using JavaScript, responsive styling, and clearer user feedback.

## Features Implemented
- Live project search that filters project cards while the user types
- Time-based greeting shown dynamically when the page loads
- Dark/light theme toggle with `localStorage` persistence
- Contact form validation with friendly success and error messages
- Responsive layout for desktop, tablet, and mobile screens
- Simple transitions and hover effects for smoother interaction

## How to Run Locally
1. Download or clone the repository.
2. Open the `assignment-2` folder.
3. Open `index.html` in a web browser.
4. Test the interactive features:
   - type in the project search box
   - toggle the theme
   - try the contact form with valid and invalid input
